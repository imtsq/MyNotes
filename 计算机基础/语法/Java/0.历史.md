# Java编译方式

Java是混合型，不是解释型也不是编译型

将java文件整体编译为一个 class文件（字节码文件），然后按行（这里就是解释型的特性）交给设备运行，但不是直接在设备运行，是通过设备的虚拟机运行

# JDK

- Jvm
  - java虚拟机

- 核心类库

- Java开发工具

  - javac编译工具

  - java运行功能

  - jdb调试工具

  - jhat内存分析工具

  - ......

# JRE

Java的运行环境，有JVM，核心类库，运行工具，相当于是jdk的缩小版，有些开发工具中的工具比如javac是没有的只有运行所需的环境
